# RoboFriends

This is a react app following Andrei Neagoie react+redux tutorial.

[Tutorial APP Demo](http://asilverio.com/demos/robofriends/index.html?target=_blank)

## Installation

To run the project:

```bash
npm install
npm start
```
`npm build` to build production app.
