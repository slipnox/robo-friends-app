# RoboFriends

This is a react app following Andrei Neagoie react+redux tutorial.

[Tutorial APP Demo](http://asilverio.com/demos/robofriends/index.html)

## Installation

To run the project:

```bash
npm install
npm start
```
Run `npm build` to build production app.
